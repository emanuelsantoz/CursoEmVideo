Olha o favicon em cima



Temos 3 Site para implementamos favicon em nossos site

1- IconArchive é um site que já tem pronto os favicon pronto, apenas escolha o teu e utilize.

2- Favicon.cc é um site onde é possível desenhar o seu proprio favicon e exporta ele ja no formato ico.

3- Favicon.io é um site que gera favicon apartir de img ou emoj já prontos que você escolhe.
